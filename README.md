# Atividade Avaliativa - Git e GitHub

Projeto desenvolvido para a atividade avaliativa de versionamento com Git e GitHub.

## Objetivo

Desenvolver uma aplicação web simples com tela de login, validação de campos, dashboard, cadastro de usuário e controle de versões utilizando branches, commits, merges e publicação no GitHub Pages.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

## Funcionalidades

- Tela de login com tema escuro
- Validação de campos obrigatórios
- Mensagens de erro
- Mensagem de login realizado com sucesso
- Redirecionamento para dashboard
- Dashboard com mensagem de boas-vindas
- Tela de cadastro de usuário
- Controle de autenticação simples com localStorage
